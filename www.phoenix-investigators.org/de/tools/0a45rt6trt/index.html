TOOLS "ZOOM" + "TEXT DECODER"
 
Zoom + Text Decoder
Funktionsweise des Tools ZOOM:
Der Zoom wird über die Symbolleiste aktiviert.
Zoom
Ermöglicht die Vergrößerung des Bilds und die Identifizierung von im Hintergrund versteckten Angaben.
Schließen
Sättigung und Helligkeit
Durch entsprechende Positionierung dieser 2 Schieberegler können Sie den Bildkontrast erhöhen und das Bild aufhellen bzw. abdunkeln.
Funktionsweise des Tools TEXT DECODER:
Dieses Tool ermöglicht die Umwandlung von Ziffern in Buchstaben gemäß den herkömmlichen Verschlüsselungsmethoden und die Anzeige von auf der DVD versteckten Wörtern.
Der Textdecoder wird über die Symbolleiste aktiviert.
Eingabe des zu entschlüsselnden Codes
Das Tool "Text Decoder" unterstützt die Eingabe von etwa 35 Ziffern.
Entschlüsselungsmethoden
Testen Sie nach der Codeeingabe die verschiedenen Entschlüsselungsmethoden.
Anzeige der Ergebnisse
Sehen Sie sich die Ergebnisse an. Auf einer Zeile ist vielleicht ein Wort oder ein Name zu erkennen. Ist das nicht der Fall, dann versuchen Sie es mit einer anderen Methode.
Schließen
Bildlaufleiste
Verwenden Sie die Bildlaufleiste, um die Ergebnisse in der Anzeige abzurollen.
