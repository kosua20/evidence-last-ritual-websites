MOVIES TOOL: NEW FUNCTION
 
Video Decrypt
Using the VIDEO DECRYPT function:
Once installed, a new option appears in the MOVIES tool: VIDEO DECRYPT (at the bottom of the movie inventory bar).
Once activated (by clicking on DRAG), you will be able to take the movie thumbnails and move them onto the grid.
Puzzle assembly
Move the thumbnails from the inventory and place them on the grid's magnetic slots. As a result, you should display all of the hidden code.
 
Click on DRAG
The movie thumbnails can then be moved.
 Click on PLAY
To return to view mode (VCR).
When PLAY mode is enabled (below), you can view movies normally.
The >> and << keys let you go forward or backward frame by frame, so that you can locate the code and stop the movie on the encrypted image. You can then close the movie. Part of the code will remain visible in the inventory. This will enable you to assemble the different parts of the code on the assembly grid.
