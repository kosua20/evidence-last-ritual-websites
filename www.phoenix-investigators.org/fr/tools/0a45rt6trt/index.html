OUTILS ZOOM + TEXT DECODER
 
Zoom + Text decoder
Fonctionnement de l’outil ZOOM :
Le zoom est activé à partir de la barre d’outils.
Zoom
Permet d’agrandir l’image et trouver des indications cachées dans les fonds d’écran.
Fermer
Saturation et luminosité
En bougeant ces 2 curseurs vous pouvez contraster l’image, l’éclaircir ou l’assombrir.
Fonctionnement de l’outil TEXT DECODER :
Cet outil permet de convertir des nombres en lettres selon des méthodes classiques de cryptage et de faire apparaître des mots cachés dans le Dvd.
Le text decoder est activé à partir de la barre d’outils.
Saisie du code à décrypter
L’outil text décoder vous permet de rentrer environ 35 chiffres.
Méthode de décryptage
Une fois le code saisi, testez les différentes options de décryptage.
Affichage des résultats.
Consultez les résultats proposés. Sur une ligne, vous verrez peut-être apparaître un mot ou un nom. Sinon essayez avec autre méthode.
Fermer
Barre de défilement
Utilisez l’ascenseur pour faire défiler tous les résultats.
