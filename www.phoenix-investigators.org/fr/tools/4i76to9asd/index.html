Rootkit
Fonctionnement de l’outil ROOTKIT :
Le rootkit est une porte d’entrée dérobée sur la machine du Phoenix.
Login
Tapez ici le login du Phoenix quand vous l’aurez trouvé.
Password
Tapez ici le mot de passe quand vous l’aurez trouvé.
.
Valider
Une fois le login et le mot de passe tapés, cliquez sur la touche >>> pour valider. Attendez ensuite les instructions.
