OUTIL MOVIES : NOUVELLE FONCTION
 
l'option Video Decrypt
Fonctionnement de la fonction VIDEO DECRYPT :
Une fois installée, une nouvelle option apparaît dans l’outil MOVIES : l’option VIDEO DECRYPT (en bas de la barre d’inventaire des films).
Une fois activée (par un clic sur DRAG), elle vous permet de prendre les vignettes des films et de les déplacer sur une grille.
Montage du puzzle
Déplacez les vignettes hors de l’inventaire et placez les sur les emplacements aimantés de la grille. Vous devriez ainsi faire apparaître l’intégralité du code caché. .
 
Cliquez sur DRAG
Les vignettes des films peuvent alors être déplacées.
 Cliquez sur PLAY
Pour repasser en mode visionnage (magnétoscope)
Quand le mode PLAY est activé (ci-dessous), vous pouvez visionner les films normalement.
Les touches >> et << vous permettent d’avancer ou de reculer image par image afin de repérer le code et d’arrêter le film sur l’image cryptée. Vous pouvez alors refermer le film. La partie du code restera visible dans l’inventaire. Cela vous permettra d’assembler les différentes parties du code sur la grille de montage.
