OUTIL TRACER
 
Tracer
Fonctionnement de l’outil TRACER :
Comme les autres outils, le Tracer est activé à partir de la barre d’outils.
Le tracer permet d’afficher les codes d’entrée / sortie utilisés par les personnes qui tentent de se connecter à votre machine.
Il permet de repérer trois clés utilisées par le Phoenix. Ces 3 clés devraient me permettre de localiser son adresse IP.
 
Repérage du code
Repérez les 4 chiffres récurrents ; il s’agit d’une clé utilisée par le Phoenix pour se connecter à votre machine. Attention ce chiffre n’apparaît pas forcément sur chaque ligne. N’hésitez pas à relancer de nouvelles captures si les chiffres n’apparaissent pas de manière évidente.
 
Capture
Quand vous repérez une connexion qui part de l’Ecosse, cliquez sur Capture.
 Transfert
Une fois que vous avez trouvé une première clé, tapez la en dessous puis identifiez la suivante en renouvelant l’opération. Quand vous aurez trouvé les 3 clés, cliquez sur Transfert pour les valider.
