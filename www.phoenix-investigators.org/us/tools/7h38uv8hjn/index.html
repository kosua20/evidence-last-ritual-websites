Tracer
Using the TRACER:
As with the other tools, the Tracer is activated from the toolbar.
The Tracer displays the input/output codes used by people attempting to log into your computer.
It can be used to retrieve three keys used by the Phoenix. The three keys should help me to locate his IP address.
 
Code identification
Identify the four recurring numbers; it is the key used by the Phoenix to log into your machine. Caution: the number does not necessarily appear on each line. Do not hesitate to capture other connections if you have trouble picking out the code.
 
Capture
When you identify a connection originating from Scotland, click on Capture.
 Transfer
Once you have found the first key, type it underneath and then identify the next key using the same procedure. When you have found the three keys, click on Transfer to validate them.
